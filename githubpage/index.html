Druckerverwaltungstool

Eine webbasierte Anwendung zum Verwalten von Druckern und deren Zubehören

github.com/implod3
View on GitHub Download .zip Download .tar.gz

Inspiration

Da das Verwalten von Druckern, Tonern und sonstigem schnell unübersichtlich, zeitintensiv und ungenau wird, wurde dieses Druckerverwaltungstool entwickelt.

Ziele

Die Webapplikation...

  • stellt die einfache Übersicht über alle Drucker, Zubehöre, Lieferanten und Lieferungen sicher
  • gibt alle Tonerstände in Echtzeit aus
  • prüft, ob Toner unautorisiert getauscht wurden
  • erstellt Warnungen, wenn Bestände von Druckerzubehören leer werden

Systemanforderungen

Das Verwaltungstool benötigt...

  • eine SQL-Datenbank auf dem gleichen Server
  • eine SNMP-Schnittstelle. In phpinfo(); muss diese installiert sein
